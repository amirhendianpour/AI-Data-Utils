# Dataset Splitter

این اسکریپت پایتونی برای تقسیم یک مجموعه‌داده‌ی تصویری به سه بخش `train`، `valid` و `test` استفاده می‌شود. نسبت تقسیم به‌صورت پیش‌فرض ۷۰٪ برای آموزش، ۲۰٪ برای اعتبارسنجی و ۱۰٪ برای تست است.

## ویژگی‌ها
- تقسیم تصاویر به سه بخش `train`، `valid` و `test`.
- حفظ نام کلاس‌ها در خروجی.
- امکان تغییر نسبت تقسیم داده‌ها.
- قابلیت پردازش تصاویر با فرمت‌های `jpg`, `jpeg`, `png`.
- استفاده از `shutil` برای کپی فایل‌ها به مسیرهای جدید.

## پیش‌نیازها
- پایتون ۳
- دسترسی به کتابخانه‌های `os` و `shutil`

## نحوه استفاده

1. مسیر مجموعه‌داده‌ی ورودی را در متغیر `dataset_dir` مشخص کنید.
2. مسیر خروجی را در `output_dir` تنظیم کنید.
3. نسبت تقسیم داده‌ها را در متغیرهای `train_ratio`, `valid_ratio` تنظیم کنید.
4. اسکریپت را اجرا کنید:

```bash
python split_dataset.py
```

## ساختار پوشه‌های خروجی

پس از اجرا، پوشه خروجی به شکل زیر خواهد بود:
```
output_dir/
├── train/
│   ├── class_1/
│   ├── class_2/
│   ├── ...
├── valid/
│   ├── class_1/
│   ├── class_2/
│   ├── ...
└── test/
    ├── class_1/
    ├── class_2/
    ├── ...
```

## نکات مهم
- پوشه‌ی ورودی باید شامل زیرپوشه‌هایی باشد که هرکدام نشان‌دهنده‌ی یک کلاس هستند.
- نام کلاس‌ها بدون تغییر در مسیر خروجی حفظ می‌شوند.
- اگر پوشه خروجی وجود نداشته باشد، به‌طور خودکار ساخته خواهد شد.
